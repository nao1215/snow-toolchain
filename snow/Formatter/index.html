Module Formatter

SNOW Code Formatter

This module implements a code formatter for the SNOW programming language, similar to go fmt. It takes parsed AST and pretty-prints it with consistent formatting rules.

Formatting Rules:

  • Consistent indentation (2 spaces)
  • Proper spacing around operators
  • Consistent line breaks
  • Standardized function/type definitions
type fmt_context = {
